- Tags

new:

> cmd + t

close:

> cmd + w

switch:

> cmd + 1/2/3
> cmd + left/right

full screen:

> cmd + enter

search:

> cmd + f

- Screen split

horizontally:

> cmd + d

vertically:

> ^ + cmd + d

> ctrl + u

> ctrl + a

move to end:

> ctrl + e

> ctrl f/b

previous command:

> ctrl p

> ctrl + r

> ctrl + d

delete previous:

> ctrl + h

delete after:

> ctrl + w

delele till the end:

> ctrl + k

> ctrl + t

> cmd + r
